# Checklist SEO Local - FormaFlow

Este checklist apunta a mejorar posicionamiento para busquedas como:
- Formularios
- Forma Flow
- FormFlow
- Formularios Muni
- Formularios San Carlos

## 1) Google Search Console

1. Verificar propiedad de dominio exacta: `https://formaflow-sancarlos.web.app/`.
2. Enviar sitemap: `https://formaflow-sancarlos.web.app/sitemap.xml`.
3. Solicitar indexacion manual de:
   - Home: `https://formaflow-sancarlos.web.app/`
   - Landing SEO: `https://formaflow-sancarlos.web.app/formularios-municipales-san-carlos/`
4. Revisar el reporte de cobertura y corregir cualquier URL excluida importante.
5. Revisar Core Web Vitals y priorizar mejoras en URLs con peor estado.

## 2) Google Business Profile (si aplica institucionalmente)

1. Vincular la web oficial.
2. Usar descripcion con terminos locales:
   - Formularios municipales
   - San Carlos
   - Tramites digitales
3. Publicar novedades con enlace a la landing SEO.

## 3) Enlaces institucionales (autoridad local)

1. Pedir enlace desde sitio oficial municipal hacia:
   - Home de FormaFlow
   - Landing SEO local
2. Solicitar enlace desde secciones de Tramites, Prensa o Atencion Ciudadana.
3. Mantener consistencia de nombre: FormaFlow (Forma Flow / FormFlow).

## 4) Contenido continuo (cada 2-4 semanas)

1. Crear una nueva pagina corta por tramite frecuente (ejemplo: "Solicitud de ... en San Carlos").
2. Cada pagina debe tener:
   - Titulo unico
   - Meta descripcion unica
   - Canonical propio
   - Enlace al portal
3. Actualizar sitemap cuando se agreguen nuevas paginas.

## 5) Seguimiento de resultados

1. Medir impresiones, clics y posicion media por consulta en Search Console.
2. Priorizar consultas con alta impresion y bajo CTR.
3. Ajustar titulos/descripciones segun resultados cada mes.
